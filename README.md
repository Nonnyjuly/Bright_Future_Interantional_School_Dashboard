Bright Future International School Performance Dashboard
Overview

This project presents an interactive Student Performance Dashboard for Bright Future International School, analysing academic performance and student engagement from Term 1 to Term 3.
The dashboard provides data-driven insights to support school management and educational decision-making.

Key Features

*Overview KPIs: total students, average score, highest score, and total absence days
*Subject performance analysis across Mathematics, English, Biology, Chemistry, and Physics
*Monthly attendance and absenteeism trends
*Top-performing students identification
*Student demographics, gender distribution, and career aspirations
*Extracurricular activities and part-time work insights

Tools & Technologies

*Power BI
*Excel
*SQL (data preparation)

Purpose

The dashboard demonstrates practical skills in data analysis, visualisation, and business insight generation, showcasing how educational data can be transformed into actionable insights.
